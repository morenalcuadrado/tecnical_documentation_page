Tip 5: Smartphone Security:
Smartphone Security

Screen Lock

  • The screen lock is the first line of defence. Be sure to set up a lock method, such as a PIN, pattern or password. Iphones offer options such as Face ID or Touch ID for quick and secure access. Ex.:
    • Good = Face ID. NOT GOOD block pin like = 1234

Two-Factor Authentication (2FA)

  • Enable two-factor authentication on your accounts. This adds an extra layer of security by requiring a unique code sent to your phone to access your accounts. Ex.:
    • Phone code = *************. Email code = *************.
